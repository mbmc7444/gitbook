# 기본개념

#### Javascript는 사용자와 상호작용한다

사용자가 어떤 코드를 작성하면 그 코드에 맞게 웹페이지를 만들어주는것

```text
<h1>1</h1>

<script>
document.write(1)
</script>
```

둘다 웹브라우저 공간에서는 1이 출력이 된다

```text
<h1>1+1</h1>

<script>
document.write(1+1)
</script>
```

이렇게 할경우 html 은 1+1 그대로 출력하는 반면에, javascipt는 2가 출력이된다

{% hint style="info" %}
html 정적인 언어

javascript 동적인 언어
{% endhint %}
