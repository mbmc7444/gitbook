# 조건문

#### 조건문이란 어떤 코드를 실행하기에 앞서 조건에 부합되는지 확인하 코드이다.

```text
<h2>IF-true</h2>
<script>
  document.write("1<br>");
  if(true){
    document.write("2<br>");
  } else {
    document.write("3<br>");
  }
  document.write("4<br>");
</script>

<h2>IF-false</h2>
<script>
  document.write("1<br>");
  if(false){
    document.write("2<br>");
  } else {
    document.write("3<br>");
  }
  document.write("4<br>");
</script>
```
