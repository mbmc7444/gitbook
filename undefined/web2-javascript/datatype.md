{% tab title="숫자\(Number\)" %}
  
 숫자에서 가장중요한 것은 연산이다.

산술연산자 : 왼쪽과 오른쪽 값을 \(+,-,\*,/\) 하여, 하나의 값을 만든 

```text
1+1 // 더하
2-1 // 빼
2*4 // 곱하
6/2 // 나누기

산술연산자
왼쪽값과 오른쪽값을 (+,-,*,/) 하여 ,하나의 값을만든다
```

{% tab title="문자\(String\)" %}
string 중에 유용한 Methods가 몇가지가 있다

```text
'Hello world'.toUpperCase() //소문자를 대문자로 변환시켜줌
'Hello world'.indexOf(‘o’) //몇번째에 글자가 존재하는지 알려준다
'Hello world'.trim() // 공백을 없애준다
```
